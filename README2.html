1. Clone Globulator using git.

git clone https://github.com/srhartono/Globulator

If you do not have git, go to http://git-scm.com/book/en/Getting-Started-Installing-Git for instructions how to set up git

2. Download example files


Download DIC_example.tif and RG_example.tif from https://www.dropbox.com/sh/mwgupokswkiqi6d/_cwGVzl7Xg/
Gunzip each file (e.g. gunzip DIC_example.gz)
Put DIC_example.tif in PUT_SLIDES_HERE/DIC/
Put RG_example.tif in PUT_SLIDES_HERE/RG

3. Run start.pl in main folder

perl start.pl

4. Results

1. ImageJ will start running and process DICexample.tif and RGexample.tif files. It's recommended to let the program run and not use the computer in the meantime. ImageJ will turn each image into black and white mask and each will produce coordinate list.

2. After imageJ is done, it will close and globulator.pl, summarizer.pl, and draw.pl will start to run. The result folder name will be shown.

3. Finally, you can also view the resulting map based on imageJ resulting coordinates. If you do not have R and draw.pl failed to run, it is okay because the resulting globule/crescent coordinates is still in the result folder, but you won't be able to see the map.
